+ Platform: Linux
+ Compiler: gcc

This software is c90-compliant.

## Compiling

You may use the provided Makefile to build it.

Open a command line prompt and go to the software directory.

``$ cd /path/to/radix/sort/project/``

Then, run the make program:

``$ make``

To run the program:

``$ make run``

or

``$ ./run.sh radix-sort``
